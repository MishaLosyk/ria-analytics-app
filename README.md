Analytics app
===============================
    
quick start
===========

**Install modules**
```sh
$ npm install
```

**Start app:**
```sh
$ node ./index.js
```

**Address**

Web application will be available on "http://localhost:8081"

**Front-end**

You can find front-end app (vue.js) in folder **"front-main"**.
Please check there readme to launch UI application.
